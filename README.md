## 🛠 개발 환경 구성

### 1️⃣ 개발 버전

| 항목        | 버전                       |
| ----------- | -------------------------- |
| Node.js     | 22.13.1                    |
| npm         | 10.9.2                     |
| Next.js CLI | 15.2.5 (`create-next-app`) |

```bash
# Node.js & npm 설치
nvm install 22.13.1
nvm use 22.13.1
nvm alias default 22.13.1

npm install -g npm@10.9.2

# Next.js 프로젝트 생성
npx create-next-app@15.2.5
```
